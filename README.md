### 🚀 Proyectos Destacados

#### 📁 [Sistema de Autogestión Web (EAMSEP)](https://github.com/Alito94/sitioEamsep)
Desarrollo de una plataforma web completa de autogestión. El mayor desafío técnico de este proyecto fue diseñar e implementar una arquitectura backend moderna (Node.js/Express) capaz de conectarse e interactuar de forma segura con una base de datos relacional heredada (Visual FoxPro), permitiendo la migración y conectividad de sistemas críticos.

#### 📁 [Sistema de Gestión Integral para Gimnasios](https://github.com/Alito94/Sistema_de_control_para_gimnasio)
Plataforma Fullstack para administrar la operativa de un centro deportivo. Incluye control de acceso en tiempo real por WebSockets (molinete inteligente), bot de WhatsApp automatizado para cobranzas y gestión de caja. Empaquetado como aplicación de escritorio portable con Node.js y MySQL embebidos.

#### 📁 [API RESTful - Gestión Escolar](https://github.com/Alito94/Gestor_escolar)
Sistema backend orientado a reducir la carga administrativa docente. Automatiza inscripciones masivas mediante el procesamiento de archivos Excel y consultas SQL dinámicas, con una base de datos flexible adaptada a la realidad del aula.
